## Loops/Iterative Statements

Loops -> Do something until a certain conditions fails
In C,

- while
- for
- do-while

### while

```
    while(condition){
        // do something
    }
```

### for

```
    for(int i=initial_value; condition ; increment or decrement of i){
        // do something
    }
```

### do-while

the statement will execute atlest once

```
do{
    // do something
}    while(condition)
```
